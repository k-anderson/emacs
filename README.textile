h2. Packages

h3. Commons

The commons package has definitions of useful functions and hooks to be used in every mode.

Defined keys:
* \C-c: -> 'uncomment-region
* \C-c; -> 'comment-region
* \C-c\C-c -> 'comment-region
* \C-F -> 'commons/indent-whole-buffer
* \C-A -> 'mark-whole-buffer

h3. "Ido":http://www.emacswiki.org/emacs/InteractivelyDoThings

This package is just awsome! But if you don't want to use it, just remove "ido" from the *load-cfg-files* in .emacs file.

h3. Theme

Currently I'm using the "irblack":http://github.com/dz/ir_black_emacs theme if you want to change that just edit configs/theme.el.

h3. Editor

This package contains the settings for the emacs editor itself like font setup, remove the menu, etc. I'm using the Monaco font, if you don't have it the TTF is on the vendor/fonts directory.

h3. "yasnippet":http://code.google.com/p/yasnippet/

Another useful package that I use. Use <TAB> to use it. If you don't want to use this remove the "yasnippet" from the *load-cfg-files*.

h3. "Flymake":http://www.emacswiki.org/emacs/FlyMake
