h2. Packages

My emacs configurations files are organized into small packages.

If you don't want to use some of the packages, just remove the "package-name" from *load-cfg-files* function.

h3. Commons

The commons package has definitions of useful variables, functions and hooks that are used on all of the other packages.

Useful key bindings:
* *\C-c;* and *\C-c \C-c*  -> Comments a selected region.
* *\C-c:* -> Uncomment a selected region.
* *\C-F* -> Indents the whole buffer.
* *\C-A* -> Selects everything in a buffer.

h3. Ido

This package is just awsome!

Useful key bindings:
* *\C-x \C-f* -> Finds a file using Ido.
* *\C-x b* -> Switches buffers using Ido.

References:
"http://www.emacswiki.org/emacs/InteractivelyDoThings":http://www.emacswiki.org/emacs/InteractivelyDoThings

h3. Theme

The theme package sets the current used theme. I'm using "irblack":http://github.com/dz/ir_black_emacs theme but if you want to change that just edit *configs/theme.el*.

h3. Editor

This package contains some useful settings for emacs editor, like font-face, font-size, menu appearance, etc. The current setting is using *Monaco* font, if you don't have it installed in your system, use the TTF that is placed on *vendor/fonts* directory.

h3. Yasnippet

Another useful package that I use. With this activated you can use *<TAB>* to autocomplete using some pre-defined templates. They are placed at *vendor/yasnippet/snippets*

Useful key bindings:
* *<TAB>* -> Autocomplete with yasnippet.

References:
http://code.google.com/p/yasnippet/
http://www.emacswiki.org/emacs/Yasnippet
